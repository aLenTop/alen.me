# alen.me
alen.me - скрипт для сайта nekto.me, имеющий ряд полезных функций.

- голосовой движок = распознавание речи для быстрого управления чатом командами скип, стоп и старт.
- эффекты изменения голоса = наложение фильтров на ваш микрофон и на голос собеседника, есть режимы рации, робота, телефона и другие.
- настройка питча = изменение высоты и скорости речи собеседника.
- встроенные игры = лаунчер с играми в стиле windows 98, включая динозаврика, сапера, змейку, тетрис и 2048, чтобы не скучать при поиске, их штук 10.
- таймер доступа = ограничение времени нахождения на сайте с автоматической блокировкой на перерыв.
- статистика общения = запись истории ваших диалогов и времени разговоров.
- настройки аудио = детальная регулировка громкости, самопрослушивание микрофона и регулировка его усиления.
