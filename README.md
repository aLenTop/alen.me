<summary>🔍 Техническое описание для поиска / SEO Keywords</summary>

**alen.me** — многофункциональный скрипт (userscript для Tampermonkey) для анонимного голосового чата **nekto.me** (некто ме / некто.ми).

* **Голосовое управление чатом (Speech Recognition):** Распознавание русской речи, голосовые команды для управления поиском: *старт*, *стоп*, *скип* (skip, stop, start). Система анти-зависания (watchdog) для стабильной работы микрофона и Web Speech API в Chrome.
* **Изменение голоса в реальном времени (Voice Changer / Pitch Shift):** Изменение питча (высоты и скорости) речи собеседника. Звуковые эффекты для микрофона и входящего звука через Web Audio API: рация, телефон, робот, эхо, реверберация (пещера, стадион), фленджер, винил, lofi, деградация звука.
* **Встроенные игры (Offline HTML5 Games):** Окна в стиле Windows 98. Работают без интернета через изолированный `iframe srcdoc` (обход CORS). Игры: Динозаврик Chrome (T-Rex runner), Сапер (Minesweeper), Змейка (Snake), Тетрис / Блоки (Tetris), 2048, Флэппи (Flappy Bird), Арканоид (Breakout), Саймон, Пинг-понг, Камень-ножницы-бумага, Memory, Кроты, Тест реакции, Крестики-нолики.
* **Контроль времени и статистика:** Лимит времени на один диалог, таймер ограничения доступа к сайту (блокировка на перерыв по абсолютному времени), подробная статистика звонков и длительности разговоров.
* **Настройка аудио:** Усиление микрофона (mic gain), самопрослушивание, независимая регулировка громкости чата и оповещений, глушение звука (mute) собеседника или микрофона кнопками.
* **Удобство:** Горячие клавиши (хоткеи с модификаторами Shift/Alt/Ctrl), кастомные темы оформления интерфейса (темная, океан, лайм, неоновый фиолет), индикатор пинга (RTT через WebRTC peerstats).
